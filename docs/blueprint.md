# **App Name**: AUTOGC

## Core Features:

- Landing Page with Lead Capture: A marketing-focused landing page featuring course details, testimonials, and a registration form for potential trainees to submit their email addresses.
- Trainee Dashboard: A secure dashboard accessible only to approved trainees, featuring a clean sidebar navigation for course modules and embedded video lessons.
- Google Drive Video Integration: Securely embed Google Drive videos within the Trainee Dashboard, allowing streaming of course content directly from Google Drive using shared links.
- Resource Downloads: Provide a section within each course module for trainees to download relevant PDF attachments and engineering blueprints.
- Admin Panel with Secure Access: A hidden admin panel accessible via a specific URL, secured by a 'secret code' login, ensuring only authorized personnel can manage the platform.
- Trainee Management System: An admin interface to view a table of all emails submitted via the landing page, with 'Approve' and 'Reject' buttons to manage trainee access.
- Content Management: A user-friendly admin interface form to update Google Drive video links that appear in the Trainee Dashboard, ensuring content is always up-to-date.

## Style Guidelines:

- Primary color: Deep Blue (#304D63) to reflect a sense of stability, trust, and professionalism.
- Background color: Light Grey (#D9D9D9) to create a clean and neutral backdrop, ensuring readability and focus on content.
- Accent color: Slate Grey (#737373) to provide subtle contrast, used for interactive elements, and CTAs, and highlighting key information.
- Body and headline font: 'Inter', a grotesque-style sans-serif, will provide a modern and neutral look suitable for both headlines and body text.
- Use professional, engineering-themed icons throughout the platform, with a focus on clarity and relevance.
- Maintain a clean, well-structured layout across all interfaces to ensure ease of navigation and a professional user experience. Utilize clear visual hierarchy to guide users through the content.
- Implement subtle, purposeful animations, such as transitions on button hover and loading states, to enhance user engagement without being distracting.
